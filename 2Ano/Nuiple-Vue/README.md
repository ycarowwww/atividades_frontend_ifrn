# Nuiple

Trabalho Final da matéria de **Design Web** do IFRN - Campus Natal-Central. O site é um pequeno "Gerenciador de Músicas" chamado **Nuiple**, que permite o usuário adicionar músicas a uma fila e executá-las localmente sem precisar de uma conta. Possui pouquíssimas músicas por padrão devido a motivos de armazenamento. Para executar esse programa, prossiga com os passos abaixo:

1. Tenha o **NodeJS** instalado na sua máquina.
2. Baixe os arquivos e execute os comandos no CMD dentro da pasta do Nuiple:
```sh
npm install
npm run dev
```
3. Entre no link local que aparecerá no CMD.
